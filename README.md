## Material CMS

----

### Ultra fast CMS with material design, reusable widgets.

**🧠 Developer Ready**
**🐳 Dockerfile**
**💻📲 Mobile Ready**
**🔖 OnePage Navigation**
**⚫️ Material Design**

You can build a fully working content management system in 10 Minutes and modify your website with fullpage, materialized content.

![Screenshot](screenshot.jpg)

### Installing instructions

````
# clone repository
git clone https://github.com/felixlberg/fullpage-cms
# cd into the new project
cd fullpage-cms
# Install dependencies
npm install
# Add an admin user to the admin group; prompts for password
node app.js apostrophe-users:add admin admin
# Go go go!
mpm start
````
